Linux Distributions

Information, reviews about various Linux Operating Systems (Linux distros).

Contribute

  • Fork this project
  • Clone your fork to your computer
  • Copy the file distro-name.md to the distro you’re gonna add. Example : bash cp distro-name.md linux-mint.md
  • Edit the newly copied file and add the details. Sample File, Ubuntu
  • After making necessary changes, add the file to git, commit and push. Example : git add linux-mint.md git commit linux-mint.md -m 'Added information about Linux Mint' git push
  • Make a pull request to this repository
  • That’s it !
