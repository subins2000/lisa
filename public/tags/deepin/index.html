Deepin OS

DEEPIN OS Deepin, stylized as deepin was formerly known as Linux Deepin and Hiweed Linux. What sets Deepin apart from other Linux Distributions is the is the strategic partner of CodeWeavers who packaged a TidBit into Deepin named “CrossOver”. CrossOver is a Microsoft Windows compatibility layer for Linux. This layer enables Windows-based applications to run on Linux OSs.
