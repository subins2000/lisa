Manjaro Linux

MANJARO LINUX Manjaro Linux, is an open source operating system for computers. It is a distribution of Linux based on the Arch Linux distribution. Manjaro Linux has a focus on user friendliness and accessibility and the system itself is designed to work fully “straight out of the box” with its variety of pre-installed software.
